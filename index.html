Sobre Mim

Vinda do Suporte de Infraestrutura, trago comigo a resiliência e a visão analítica necessária para resolver problemas complexos sob pressão. Hoje, como desenvolvedora Full Stack, foco na construção de sistemas eficientes e escaláveis.

No Back-End, domino a stack Java com Spring Boot e MySQL, criando APIs REST seguras e performáticas. No Front-End, utilizo React e Bootstrap para dar vida a interfaces modernas e responsivas, sempre com foco na experiência do usuário.

Atualmente cursando Análise e Desenvolvimento de Sistemas, sigo aprimorando conhecimentos em Cloud (Azure) e automação de processos, unindo o melhor dos dois mundos: código limpo e infraestrutura sólida para criar soluções digitais completas.
